How to Play

1. Create or Join

Enter your name, select a game, and create a room. Share the 4-letter room code with friends, or join their room.

2. Wait in Lobby

The host can start the game once at least 2 players have joined. Use the lobby chat to communicate!

3. Battle!

Compete in real-time. Each game has unique rules - read the instructions and do your best!

4. Win & Rematch

Highest score wins! Return to the lobby for a rematch or try a different game.
